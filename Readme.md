Клиентская программа для клиент-серверной системы обработки данных

Компиляция:
make all
Или отдельно:
make release
make debug

Очистка:
make clean

Создание конфигурационного файла:
sudo mkdir -p /.config
echo "user:P@ssW0rd" > /.config/vclient.conf
//Формат логин:пароль

Запуск сервера:
chmod +x server_static
./server_static -T int16_t -H SHA224 -S server

Запуск клиента:
./client -a 127.0.0.1 -i vectors.bin -o results.bin

Параметры командной строки:
-a Адрес сервера, Обязательный
-i Входной файл с векторами, Обязательный
-o Выходной файл для результатов, Обязательный
-p Порт сервера, Не обязательный
-c Файл конфигурации, Не обязательный
-h Показать справку, Не обязательный
